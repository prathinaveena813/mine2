Using NumPy & Pandas

NumPy helps you work with fast numerical arrays. Pandas helps you clean, transform, summarize, and explore tabular data like CSV files, Excel sheets, and database exports.

Python Arrays DataFrames Analytics

NumPy

NumPy is used for numerical computing. It is best for arrays, mathematical operations, statistics, and efficient calculations.

  • Create arrays
  • Calculate mean, median, standard deviation
  • Perform vectorized operations

Pandas

Pandas is used for structured data. It makes it simple to load, filter, group, merge, and analyze datasets.

  • Read CSV or Excel files
  • Clean missing values
  • Group and summarize data

Basic Example

A small analytics workflow usually starts by loading data, checking it, cleaning it, and summarizing it.

import numpy as np
import pandas as pd

data = {
    "product": ["A", "B", "C", "D"],
    "sales": [120, 90, 150, 80],
    "profit": [30, 20, 45, 12]
}

df = pd.DataFrame(data)

average_sales = np.mean(df["sales"])
top_product = df.sort_values("profit", ascending=False).iloc[0]

print("Average sales:", average_sales)
print("Top product:", top_product["product"])

Simple Analytics Workflow

1. Load: Import data using pd.read_csv() or pd.read_excel().
2. Inspect: Use head(), info(), and describe().
3. Clean: Handle missing values, duplicates, and incorrect formats.
4. Analyze: Use filtering, grouping, sorting, and NumPy calculations.
5. Share: Export results or visualize them with charts.
NumPy + Pandas = a practical foundation for everyday data analytics.
